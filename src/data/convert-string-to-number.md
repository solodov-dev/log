title: Convert string to number
date: dec 4, 2020
tags: [js]
---

Convert string to number in javascript:

```js
// To number
const num = Number(str);

// A concise way
const num = +str;

// Will parse up to the first non-digit character
const num = parseInt(str);

// Parse float number
const num = parseFloat(str);
```
